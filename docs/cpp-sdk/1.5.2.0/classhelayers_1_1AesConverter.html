Detailed Description

A class that allows to perform AES decryption of AES-encrypted blocks, while the AES secret key, the blocks to be decrypted and the decryption results are encrypted under FHE.

FHE-encrypts the given AES ciphertext blocks, then performs AES decryption in counter (CTR) mode of operation under FHE.

The result is the AES-decrypted blocks, encrypted under FHE. A vector of batches of ciphertexts, where slot i of ciphertext j of batch k contains the j'th value of the i'th block of the k'th batch. Result size depends on the number config.

Parameters
ivThe counter mode initialization vector (IV), should contain 16 bytes (128 bits).
aesCiphertextsA vector of AES ciphertext blocks. Each block should contain 16 bytes (128 bits).
configThe configuration of the number to extract.
