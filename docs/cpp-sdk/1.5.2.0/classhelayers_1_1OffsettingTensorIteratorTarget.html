Detailed Description

An iterator target that adds a fixed offset to one dimension.

For example, if the source iterator is iterating over the range (0,0),(0,1),...(4,5) And this object is configured to offset dim 1 by 3, the target will iterate over (0,3),(0,4),...(4,8)
