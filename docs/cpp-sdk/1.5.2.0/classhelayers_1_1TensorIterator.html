◆ nextFromDim()

bool helayers::TensorIterator::nextFromDim ( DimInt  dim)

Advance iterator starting from specified dim.

If it is a firstOrder iterator, it doesn't move dimensions 0...dim-1, Otherwise it doesn't move dimensions dim+1,...lastDim. Returns false if no next element in the movable dimension range. In case it returns false it also rewinds the movable dimensions.
