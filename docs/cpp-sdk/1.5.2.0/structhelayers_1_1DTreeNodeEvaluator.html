Detailed Description

An abstract class holding the functions necessary to evaluate an encrypted decision tree model (DTree).

An implementation of a non-abstract child of this class should be provided by the user to enable the classification process. The class' representation will differ depending on the required evaluation algorithm. See DTreeDefaultEvaluator and BinaryNodeEvaluator for examples of possible implementations of this abstract class.

Returns the result of the comparison "c <= t".

Let u be a non-leaf node of the decision tree having a threshold t_u, and let f_u be the corresponding feature provided by the user. This function will be used to evaluate the condition "t_u <= f_u" by calling compare(t_u, f_u), and its return value will be added to the scores of all leaves in the left sub-tree of u, using addScore() function.

Parameters
cThe first CTile to compare.
tThe second Tile to compare.

Reimplemented in helayers::DTreeDefaultEvaluator, and helayers::DTreeMockupSignEvaluator.

virtual CTile helayers::DTreeNodeEvaluator::negateCompRes ( const CTile &  compRes) const
pure virtual

Negates a comparasion result which was returned by compare() function.

Let u be a non-leaf node of the decision tree having a threshold t_u, let f_u be the corresponding feature provided by the user and let compRes be the comparison result obtained by calling compare(t_u, f_u). This function will be used to negate the result of the comparison "t_u <= f_u", and its return value will be added to the scores of all leaves in the right sub-tree of u, using addScore() function.

Parameters
compResThe comparison result to negate.

Implemented in helayers::DTreeDefaultEvaluator, and helayers::BinaryNodeEvaluator.
