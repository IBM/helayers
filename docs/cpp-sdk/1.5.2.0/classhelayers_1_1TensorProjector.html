Detailed Description

This class is for converting flat index of a large tensor into a projected flat indices of two smaller ones.

It assumes all tensors are stored with the first order convention.

It is used for binary operators, where the result is the large tensor, and the operands are the smaller ones. For example, adding the tensors A[2,1,3] and B[2,5,1] results with C[2,5,3]. This class can be used to project a flat index x into C into the two flat indices y,z into the operands such that C[x] holds the result of A[y]+B[z]

It is thread safe.
