◆ automaticBootstrapping

bool helayers::HeConfigRequirement::automaticBootstrapping = false

Whether bootstrapping is applied automatically to prevent runtime errors resulting from executing a chain index consuming operation on a ciphertext with too small chain index.

The logic for deciding the bootstrapping points is scheme and library specific. But, in general, the bootstrappings are delyed as much as possible. This flag can only be set to true if this HeContext supports bootstrapping.

◆ bootstrapConfig

std::optional<BootstrapConfig> helayers::HeConfigRequirement::bootstrapConfig

If bootstrappable flag is set to true, this optional member can be used to set the bootstrap configuration.

If this member is empty and bootstrappable flag is true, a default bootstrap configuraition will be used.

◆ bootstrappable

bool helayers::HeConfigRequirement::bootstrappable = false

Whether CTiles created by the HeContext will be bootstrappable.

Setting this flag to true will lead to the creation of a BootsrapEvaluator object upon the initialization of the HeContext. Note that this BootstrapEvalutor creation increases the runtime of the initialization.
