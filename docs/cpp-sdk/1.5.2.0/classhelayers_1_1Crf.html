Detailed Description

a FHE implementation of Completley Random Forest.

Based on "Encrypted statistical machine learning: new privacy preserving methods" paper: https://arxiv.org/pdf/1508.06845.pdf.

void helayers::Crf::fit ( const CTileTensor &  xTrain,
const CTileTensor &  yTrain 
)

Fits the model with the given training data.

This function can be called several times on different training batches. In such a case, the result will be equivalent to calling fit once on all of the batches combined. This function assumes that the hyperparameters of this Crf object are already set.

Parameters
xTrainThe encrypted train samples.
yTrainThe encrypted train labels.
Exceptions
invalid_argumentIf the shapes of xTrain and yTrain are not valid. See also verifyValidShapes().
runtime_errorIf the the hyperparameters of this Crf object are not set.

Parameters
numTreesThe number of trees in the completely random forest

predictorDescriptionsA vector describing the predictor types (i.e. CATEGORICAL or ORDINAL) and the set of column representing them in the training samples tensor.
