◆ encryptHashSideA()

EC_POINT* helayers::er::Encryptor::encryptHashSideA ( const Hash256 &  h) const

Encrypt a 256 bit value (a hash coming from from openSSL's sha-256) by raising to the power of side A's secret key.

The operation is performed as an Elliptic-Curve product operation over the value represented as a point on the curve, and using openSSL Elliptic-Curve operations.

Parameters
hthe 256 bit input value to encrypt by raising to the power of side A's secret key.
