bool helayers::CrfLeafPlain::isLeafCorrect ( const DoubleTensor &  input,

Returns true whether the specified sample reaches this leaf.

The s-th sample is considered to reach this leaf if for each paritionSubset, k, of this leaf, there exsists exactly one column, c \in k, s.t. the s-th value of the c-th column in the input samples tensor is 1.
