Detailed Description

A class that serves as a cache of ciphertexts, i.e.

it is holding the ciphertexts in memory. This is an abstract class inherited by CtxtMemoryCache. Other inheritence may provide access to ctxts stored in places other than the memory. Each ciphertext has an id, which is the same id given to it in the circuit structure. This class is used by the circuit-runner to execute the gates of the circuit. Ciphertext may come and go as they may it may be decided to delete, cache (to disk), prefetch or load them. A cache may be responsible to (or associated with) all ciphertexts or only with someof them. For example, we may not want to associate all ciphertexts with a cache if (for example) it is a CtxtCache acting as a liaison to another server which performs some of the computation.
