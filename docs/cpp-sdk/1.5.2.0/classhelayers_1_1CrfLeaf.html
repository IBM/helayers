std::vector<std::set<DimInt> > helayers::CrfLeaf::partitionSubsets

In the completley random forest training algorithm, the internal nodes' conditions consist of "partitions" of a randomly chosen predictors.

Let p be a predictor with n_p possible values. A partition of p consists of two distinct subsets of these n_p values whose union size is n_p. Let {leftSubset, rightSubset} be a partition of a predictor p and an internal node u. A sample, s, that reaches u, will proceed to u's left child if the value of the predictor p corresponding to s belongs to leftSubset, and it will proceed to u's right child otherwise. The vector below holds all of the partition subsets corresponding to the edges in the path from the CRFtree's root to this CrfLeaf. Each partition subset is represented by a set column indexes. The s-th sample reaches this CrfLeaf if for each each partitionSubset, k, of a predictor p, there is exactly one column index, c in k, such that the s-th value of the c-th column of the input samples tensor is equal to 1.
