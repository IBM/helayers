Detailed Description

A class representing an encrpyted decision tree model which relies on one-hot encoding of the input features.

A constructor.

Parameters
heThe heContext
minInputValThe minimal input value that is required to be supported by this OneHotDTree.
maxInputValThe maximal input value that is required to be supported by this OneHotDTree.
precisionThe required precision of the one-hot encoding of the features.
maxDepthThe maximal tree depth that is required to be supported by this OneHotDTree.
Exceptions
invalid_argumentIf encoding the user input with the given parameters require more than he.slotCount() slots. This happens when round(maxInputVal - minInputVal)/precision + 1 is greater than he.slotCount().
