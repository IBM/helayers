37 /*
38 A class to describe the architecture of an NN. The architecture consists of
39 layers, each holds specification of the layer it describes. The layers are not
40 necessarily sequential, and are represented by a directed graph of connections
41 between the layers.
42 
43 The NN arch can be loaded in two options: either from a JSON file that was
44 prepared using the Keras library, or from an ONNX file.
45 
46 For instructions for how to use Keras to define a model with architecture
47 that's being supported by Helayers, see documentation in NeuralNetJsonParser.h
48 file.
49 
50 For instructions for how an ONNX file should look like in order to define a
51 model with architecture that's being supported by Helayers, see documentation in
