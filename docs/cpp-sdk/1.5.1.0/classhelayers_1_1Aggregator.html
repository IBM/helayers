Detailed Description

A class used to aggregate reports received from electrical devices.

At the end of each time period, this class can be used to generate an aggregated log which aggregates and summarizes the reports received during this period.

generates a random CTile representing an aggregated log of a specific period.

The values of the random CTile are generated using a normal distribution having the given mean and standard deviation. Note that in a more realistic scenario, an aggregated log will be created after analyzing logs received in the last time period.

meanThe mean of the normal distribution used to generate the random values.
