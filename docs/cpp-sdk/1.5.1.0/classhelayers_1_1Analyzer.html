Sets the hyperparameters of this Analyzer.

Parameters
kControls the threshold for which a periodic sum will be considered an anomlay. The greater value k gets, the more likely it is that an even will be considered as an anomaly.
alphaControls the effect of the history on the current statistics. alpha must be a value in the range [0,1]. The closer it gets to 1, the more effect the previously reported periodic sums will have on the current statistics.
Exceptions
invalid_argumentIf alpha is not in the range [0,1].
