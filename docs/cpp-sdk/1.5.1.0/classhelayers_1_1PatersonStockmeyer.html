Detailed Description

This class implements: 1) original Paterson-Stockmeyer algorithm.

2) variation on the original Paterson-Stockmeyer that accept coffecients of Chebyshev interpolant repesented with Chebyshev base.

References: 1) Chen, Hao, Ilaria Chillotti, and Yongsoo Song. "Improved bootstrapping for approximate homomorphic encryption." Annual International Conference on the Theory and Applications of Cryptographic Techniques. Springer, Cham, 2019.

2) Paterson, Michael S., and Larry J. Stockmeyer. "On the number of nonscalar multiplications necessary to evaluate polynomials." SIAM Journal on Computing 2.1 (1973): 60-66.
