Detailed Description

Evaluates a given polynomial at a given point homomorphically, using the Paterson-Stockmeyer algorithm.

References: Paterson, Michael S., and Larry J. Stockmeyer. "On the number of nonscalar multiplications necessary to evaluate polynomials." SIAM Journal on Computing 2.1 (1973): 60-66.

Chen, Hao, Ilaria Chillotti, and Yongsoo Song. "Improved bootstrapping for approximate homomorphic encryption." Annual International Conference on the Theory and Applications of Cryptographic Techniques. Springer, Cham, 2019.
