Detailed Description

A struct representing an indicator.

It can either be set to zero or one (in plaintext), or contain an encrypted value.

void Indicator::validate ( ) const

Asserts that this Indicator is valid.

An indicator is considered valid if exactly one of the following conditions hold: "isOne" flag is set, "isZero" flag is set or "val" is not empty.
