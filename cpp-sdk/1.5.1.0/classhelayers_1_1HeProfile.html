Detailed Description

Class to hold a complete HE profile of a given model run, including the requirements it takes from the HE context to be init with, the tile layout and some estimated measurements with respect to its run.

Returns partially-populated HE profile object, to be used in tests.

The HE profile will not contain valid model measures and context name, will not be on fit mode, and will have a base chain index equal to the multiplication depth of the HE config requirement.

Parameters
heConfReqThe configuration requirements from the HE library.
tileLayoutThe tile tensor layout.
modeNameThe mode name.
batchDimThe batch dimension (-1 means no batch dimension).
isModelEncryptedAn indication to whether the model weights are encrypted.
circuitOptimizationAn indication to whether to use circuit optimization when running the model.
lazyEncodingAn indication to whether the model weights should be lazy encoded.
handleOverflowAn indication to whether overflow handling should be applied.
