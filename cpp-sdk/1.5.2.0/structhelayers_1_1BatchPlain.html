Detailed Description

A struct representing one batch of a dataset represented by a DatasetPlain object.

This struct may either hold just inputs or inputs and expected outputs.

DoubleTensor helayers::BatchPlain::labels

The labels of the batch's samples.

If this BatchPlain object only holds inputs, "labels" DoubleTensor will be empty. Otherwise, it will contain the expected classes of the batch's samples.
