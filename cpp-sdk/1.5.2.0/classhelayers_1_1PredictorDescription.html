Detailed Description

A class indicating how a specific predictor is represented in the completely random forest input samples tensor.

The completely random forest model uses one-hot encoding to represent predictors. Each ordinal predictor, p, is devided into n_p possible ranges (e.g. 0-1, 1-2, ..., 9-10). Then, given a sample s, the value of p in s is represented by a one-hot vector, v_{p,s}, of length n_p. v_{p,s}[i] is 1 if the value of p in s belongs to the i-th range and 0 otherwise. Similarly, each categorical predictor, p, with m_p possible categories, is represented by a one-hot vector of length m_p. This class stores for each predictor whether it is ordinal or categorical and the set of columns used to represent it in input sample tensors that will be given to the Crf class.
