Detailed Description

A FunctionalMatrix represents a matrix without storing the actual matrix values, by providing an accessor to the matrix elements which computes the element values on the fly when called.

void helayers::FunctionalMatrix::diagonalize ( bool  _isDiagonalized = true)

Declare that the matrix is "diagonalized" Diagonalization puts the main diagonal (from [0,0]) in row 0 the 2nd diagonal (starting from [0,1]) in row 1, etc.).

So e.g. 00 01 02 03 00 11 22 33 10 11 12 13 ===\ 01 12 23 30 20 21 22 23 ===/ 02 13 20 31 30 31 32 33 03 10 21 32

Parameters
_isDiagonalizedTrue (default) if and only if digonalization is needed

bool helayers::FunctionalMatrix::isDiagonalized

True iff the matrix is indexed diagonally.
