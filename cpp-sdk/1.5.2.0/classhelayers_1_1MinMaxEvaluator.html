Computes the maximum over the specified dimension.

Stores the maximum result in "ctt" and the maximum indicators in "indicators". After the function termination, dimension "dim" of the resulting "ctt" will have a single fully duplicated tile. The maximum indicators will include 1 in the maximal slots and 0 in all other slots. In case multiple values of "ctt" are equal, the resulting indicators will contain undefined values, while the maximum result in "ctt" will be correct. In case "ctt" is fully duplicated along the specified dimension, "indicators" will contain ones in all slots and its shape will be cleared from unkowns. The input "ctt" must not contain any unused slots along "dim".

Parameters
cttThe CTileTensor to compute its maximum. The maximum result will be stored here.
indicatorsThe maximum indicators will be stored here.
dimThe dimension to compute maximum along.
gRep,fRepDetermine the accuracy of the result. Greater gRep and fRep will result with a greater accuracy on the account of slower runtime and more multiplication depth.
maxDiffThe maximum possible absolute difference between two values in "ctt" whose all coordinates in all dimensions except "dim" are equal.
Exceptions
invalid_argumentIf dimension "dim" does not exist in the input's shape.
invalid_argumentIf dimension "dim" of "ctt" contains unused slots.

Parameters

gRep,fRepDetermine the accuracy of the result. Greater gRep and fRep will result with a greater accuracy on the account of slower runtime and more multiplication depth.
maxDiffThe maximum possible absolute difference between a and b.
