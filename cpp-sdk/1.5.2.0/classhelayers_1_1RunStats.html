◆ getTotalCpuTime()

int64_t helayers::RunStats::getTotalCpuTime ( const std::map< std::string, int64_t > &  latencies,
int  slotCount,
int  chainIndexOffset = 0 
) const

Returns the total cpu time.

Returns -1 if the tracked computation is too deep for the provided latencies.

Parameters
latenciesA map which maps each operation and chain index to the expected latency when this operation is executed with the corresponding chain index.
slotCountThe number of slots in the HeContext.
chainIndexOffsetWhen computing latency, each operation that was originally performed with chain index i, will be considered as performed with chain index i-chainIndexOffset.
Exceptions
runtime_errorIf there is an operation whose latency is not included in "latencies" map.
