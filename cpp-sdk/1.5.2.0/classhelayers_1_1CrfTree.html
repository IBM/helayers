Detailed Description

A FHE implementation of a Completely Random Tree.

A set of instances of this class together create a Completely Random Forest (Crf). Based on "Encrypted statistical machine learning: new privacy preserving methods" paper: https://arxiv.org/pdf/1508.06845.pdf. This class is for internal use only.

Fits this CrfTree with the given training data.

This function can be called several times on different training batches. In such a case, the result will be equivalent to calling fit once on all of the batches combined. This function assumes that this CrfTree is already initialized.

Parameters
