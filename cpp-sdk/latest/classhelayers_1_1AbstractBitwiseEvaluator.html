Returns the hamming weight of c, considering the specified bits only.

Parameters
cThe CTile to calculate its hamming weight.
fromThe first bit index in the range of bits to calculate hamming weight for.
toThe one after the last bit in the range of bits to calculate hamming weight for. If -1, all of the bits starting from "from" will be taken.
