Detailed Description

A class with functionality related to arranging numbers into AES blocks for encryption, and to packing AES-decrypted numbers into tile tensors.

Returns a recommended default locations for numbers to be arranged in blocks to be encrypted under FHE, without considering any desired data packing.

The result is a vector of vectors of objects. The i'th object of the j'th vector corresponds to the i'th slot of the j'th original tensor.
