Detailed Description

A simple class for performing local time measures.

Example use 1: 

int timeInMicros;
{
   Timer myTimer(timeInMicros);
   . . . some code . . .
}
cout << "code above took " << timeInMicros << " microseconds".

Example use 2: 

Timer myTimer();
. . . some code . . .
myTimer().stop();
cout << "code above took " << myTimer.getDuration() << " microseconds".
