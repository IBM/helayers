Verifies the given input samples tensor is valid according to Crf's expected representation.

The input tensors will be considered valid if it contains a one hot representation of all predictors. That is, for each predictor p and sample s, exactly one of the the samples tensor's columns corresponding to p contains 1 in its s-th value.

Parameters
samplesThe input samples tensor.
predictorDescriptionsA vector describing the predictor types (i.e. CATEGORICAL or ORDINAL) and the set of column representing them in the training samples tensor.
