Contact Us#

Reporting Security Issues#

“Please do not report security vulnerabilities through public channels.”

Instead, please follow the IBM Security Vulnerability Management guidelines.

  • Please, visit our security page to learn more about the security guarantees that HElayers provide.

  • In your report please include the following information: - Type of issue (e.g. buffer overflow, SQL injection, cross-site scripting, etc.) - The configuration used and Step-by-step instructions to reproduce the issue - Proof-of-concept or exploit code (if possible) - Impact of the issue, including how an attacker might exploit the issue
