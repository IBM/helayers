pyhelayers.Crf#

class Crf#

Bases: Saveable

a FHE implementation of Completley Random Forest. Based on “Encrypted statistical machine learning: new privacy preserving methods” paper: https://arxiv.org/pdf/1508.06845.pdf.

__init__(self: pyhelayers.libs.pyhelayers_cppwrappers.Crf, he: pyhelayers.libs.pyhelayers_cppwrappers.HeContext) → None#

A constuctor. Creates an empty object. A Crf object created by this constructor cannot be trained before setting the completely random forest’s hyperparameters (e.g. by loading the Crf object from a stream or calling setHyperParams()).

Parameters:

he – An HeContext.

Raises:

RuntimeError – if random devices are not available to

produce random numbers in the used system.

set_hyper_params(self: pyhelayers.libs.pyhelayers_cppwrappers.Crf, num_trees: int, depth: int, categorical_predictors: List[Set[int]], ordinal_predictors: List[Set[int]], seed: int = -1) → None#

Sets the hyperparameters of the completley random forest.

Parameters:

  • num_trees (int) – The number of trees in the completely random forest.

  • depth (int) – The depth of each tree in the completely random forest.

  • categorical_predictors (list<set<int>>) – A list of sets of indices. Every set specifies the column indexes that will represent one of the “categorical” predictors in the training samples tensor.

  • ordinal_predictors (list<set<int>>) – A list of sets of indices. Every set specifies the column indexes that will represent one of the “ordinal” predictors in the training samples tensor.
